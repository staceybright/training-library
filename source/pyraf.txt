***********
Basic PyRAF
***********

PyRAF is a command language for running IRAF (Image Reduction and Analysis Facility) tasks in a Python like environment. PyRAF is being gradually phased out at STScI and elsewhere, but RIAs may still be called upon to work with legacy code.

.. note::

   This guide is maintained as a PDF document. :download:`Download it <pdf_guides/pyraf_training.pdf>` and get started.
